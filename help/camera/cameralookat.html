CameraLookAt()

Syntax

CameraLookAt(#Camera, x, y, z)
Description
The point (in world unit) that a camera will face.

Parameters

#Camera The camera to use.
x, y, z The position (in world unit) to point the camera at.

Return value

None.

Supported OS

All
