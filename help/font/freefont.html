FreeFont()

Syntax

FreeFont(#Font)
Description
Free the given Font, previously initialized by LoadFont().

Parameters

#Font The font to free. If #PB_All is specified, all the remaining fonts are freed.

Return value

None.

Remarks

All remaining fonts are automatically freed when the program ends.

See Also

LoadFont()

Supported OS

All
