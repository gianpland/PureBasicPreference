SplineX()

Syntax

Result = SplineX(#Spline)
Description
Returns the spline 'x' position in the world, after a ComputeSpline().

Parameters

#Spline The spline to use.

Return value

Returns the spline point 'x' position in the world, after a ComputeSpline().

See Also
