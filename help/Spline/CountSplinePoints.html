CountSplinePoints()

Syntax

Result = CountSplinePoints(#Spline)
Description
Returns the number of points in the spline.

Parameters

#Spline The spline to use.

Return value

Returns the number of points in the spline.

See Also

CreateSpline(), AddSplinePoint()

Supported OS

All
