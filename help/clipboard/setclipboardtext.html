SetClipboardText()

Syntax

SetClipboardText(Text$)
Description
Stores a string into the clipboard. If the clipboard already contains text, it will be overwritten.

Parameters

Text$ The string you want to store in the clipboard.

Return value

None.

See Also

SetClipboardImage(), ClearClipboard()

Supported OS

All
