GadgetID3D()

Syntax

GadgetID = GadgetID3D(#Gadget3D)
Description
Returns the unique system identifier of the 3D gadget.

Parameters

#Gadget3D The 3D gadget to use.

Return value

Returns the unique system identifier of the 3D gadget.

Supported OS

All
