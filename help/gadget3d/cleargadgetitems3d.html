ClearGadgetItems3D()

Syntax

ClearGadgetItems3D(#Gadget3D)
Description
Clears all the items from the specified #Gadget3D.

Parameters

#Gadget3D The 3D gadget to use.

Return value

None.

Remarks

The Gadget must be one of the following types:
- ComboBoxGadget3D()
- ListViewGadget3D()
- PanelGadget3D()

See Also

AddGadgetItem3D()

Supported OS

All
