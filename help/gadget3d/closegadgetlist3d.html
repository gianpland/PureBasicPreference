CloseGadgetList3D()

Syntax

CloseGadgetList3D()
Description
Closes the current gadget list creation and go back to the previous GadgetList.

Parameters

None.

Return value

None.

Remarks

This function is supported by the following 3D gadgets:

- ContainerGadget3D()
- PanelGadget3D()
- ScrollAreaGadget3D()

Supported OS

All
