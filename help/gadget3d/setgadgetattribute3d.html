SetGadgetAttribute3D()

Syntax

SetGadgetAttribute3D(#Gadget3D, Attribute, Value)
Description
Changes the attribute value of the specified 3D gadget.

Parameters

#Gadget3D The gadget to use.
Atttribute The attribute to set.
Value The new attribute value.

Return value

None.

Remarks

This function is available for all gadgets which support attributes:

- EditorGadget3D()
- ProgressBarGadget3D()
- ScrollAreaGadget3D()
- ScrollBarGadget3D()
- SpinGadget3D()

Supported OS

All
