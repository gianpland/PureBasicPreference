EjectAudioCD()

Syntax

EjectAudioCD(State)
Description
Eject (open) or close the current CD drive.

Parameters

State The action to take. If State = 1, the CD drive is opened, else it is closed.

Return value

None.

See Also

UseAudioCD()

Supported OS

Windows, Linux
