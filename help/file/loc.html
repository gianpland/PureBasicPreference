Loc()

Syntax

Position.q = Loc(#File)
Description
Returns the read/write pointer position in the file.

Parameters

#File The file to use.

Return value

Returns the file pointer position relative to the start of the file in bytes.

Remarks

For an example look at the FileSeek() function.

See Also

FileSeek(), Lof()

Supported OS

All
