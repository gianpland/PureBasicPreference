BuildMeshTangents()

Syntax

BuildMeshTangents(#Mesh)
Description
Automatically computes the tangent vectors for every vertices of the specified mesh.

Parameters

Mesh The mesh to use.

Return value

None.

See Also

CreateMesh(), MeshVertexTangent()

Supported OS

All
