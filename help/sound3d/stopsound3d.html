StopSound3D()

Syntax

StopSound3D(#Sound3D)
Description
Stops the specified sound (if it was playing).

Parameters

#Sound3D The sound to stop. If this value is set to #PB_All, then all sounds currently playing are stopped.

Return value

None.

See Also

PlaySound()

Supported OS

All
