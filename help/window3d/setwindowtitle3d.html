SetWindowTitle3D()

Syntax

SetWindowTitle3D(#Window3D, Title$)
Description
Changes the text which is currently displayed in the specified 3D window title bar.

Parameters

#Window3D The 3D window to use.
Title$ The new title to set.

Return value

None.

See Also

GetWindowTitle3D()

Supported OS

All
