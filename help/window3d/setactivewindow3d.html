SetActiveWindow3D()

Syntax

SetActiveWindow3D(#Window3D)
Description
Activate the specified window, which means the focus has been put on this window.

Parameters

#Window3D The 3D window to activate.

Return value

None.

See Also

GetActiveWindow3D()

Supported OS

All
