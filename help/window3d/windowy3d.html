WindowY3D()

Syntax

Result = WindowY3D(#Window3D)
Description
Returns the top position on the screen, of the given window.

Parameters

#Window3D The 3D window to use.

Return value

The top position on the screen, in pixels, of the given window.

See Also

WindowX3D()

Supported OS

All
