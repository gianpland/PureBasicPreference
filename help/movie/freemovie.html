FreeMovie()

Syntax

FreeMovie(#Movie)
Description
Frees the specified movie and all its resources.

Parameters

#Move The movie to free. If #PB_All is specified, all the remaining movies are freed.

Return value

None.

Remarks

All remaining movies are automatically freed when the program ends.

See Also

IsMovie(), LoadMovie()

Supported OS

All
