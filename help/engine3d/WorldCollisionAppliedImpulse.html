WorldCollisionAppliedImpulse()

Syntax

WorldCollisionAppliedImpulse()
Description
Fetch the applied impulse values about the collision being examined with ExamineWorldCollisions()(). ExamineWorldCollisions()() 'Contacts' parameter has to be set to #True to have this command working. The applied impulse values can be retrieved with GetX(), GetY() and GetZ().

Return value

None.

See Also

ExamineWorldCollisions(), GetX(), GetY(), GetZ()

Supported OS

All
