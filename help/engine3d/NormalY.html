NormalY()

Syntax

Result.f = NormalY()
Description
Returns the 'y' normal value at the crossed point, after RayCast(), RayCollide() or MouseRayCast().

Return value

Returns the 'y' normal value at the crossed point.

See Also

RayCast(), RayCollide(), MouseRayCast(), NormalX(), NormalZ()

Supported OS

All
