PeekB()

Syntax

Value.b = PeekB(*MemoryBuffer)
Description
Reads a byte (1 byte) number from the specified memory address.

Parameters

*MemoryBuffer The address to read from.

Return value

Returns the value of the byte.

See Also

PokeB()

Supported OS

All
