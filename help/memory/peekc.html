PeekC()

Syntax

Value.c = PeekC(*MemoryBuffer)
Description
Reads a character (1 byte in ascii, 2 bytes in unicode) number from the specified memory address.

Parameters

*MemoryBuffer The address to read from.

Return value

Returns the value of the character.

See Also

PokeC()

Supported OS

All
