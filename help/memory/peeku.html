PeekU()

Syntax

Value.u = PeekU(*MemoryBuffer)
Description
Reads an unicode character (2 bytes) from the specified memory address.

Parameters

*MemoryBuffer The address to read from.

Return value

Returns the value of the unicode character.

See Also

PokeU()

Supported OS

All
