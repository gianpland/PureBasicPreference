PokeC()

Syntax

PokeC(*MemoryBuffer, Number)
Description
Writes a character (1 byte in ascii, 2 bytes in unicode) number to the specified memory address.

Parameters

*MemoryBuffer The address to write to.
Number The value to write.

Return value

None.

See Also

PeekC()

Supported OS

All
