PeekD()

Syntax

Value.d = PeekD(*MemoryBuffer)
Description
Reads a double (8 bytes) number from the specified memory address.

Parameters

*MemoryBuffer The address to read from.

Return value

Returns the value of the double.

See Also

PokeD()

Supported OS

All
