CloseCryptRandom()

Syntax

CloseCryptRandom()
Description
Closes the cryptographic safe pseudorandom number generator that was opened with OpenCryptRandom() and frees its resources.

Parameters

None.

Return value

None.

See Also

OpenCryptRandom(), CryptRandom(), CryptRandomData()

Supported OS

All
