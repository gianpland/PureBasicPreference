UpdateTerrain()

Syntax

UpdateTerrain(#Terrain)
Description
Updates the terrain. This is needs after altering the terrain with commands like SetTerrainTileHeightAtPoint().

Parameters

#Terrain The terrain to update.

Return value

None.

See Also

CreateTerrain(), SetTerrainTileHeightAtPoint()

Supported OS

All
