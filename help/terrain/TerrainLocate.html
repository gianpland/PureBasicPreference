TerrainLocate()

Syntax

TerrainLocate(#Terrain, x, y, z)
Description
Changes the terrain absolute location in the world.

Parameters

#Terrain The terrain to use.
x, y, z The new absolute location in the world (in world unit).

Return value

None.

See Also

CreateTerrain()

Supported OS

All
