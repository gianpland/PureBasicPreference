HideMenu()

Syntax

HideMenu(#Menu, State)
Description
Hide or show the specified menu.

Parameters

#Menu The menu to hide or show.
State The new state for the menu. A value of 1 hides the menu and a value of 0 shows it.

Return value

None.

See Also

CreateMenu(), CreateImageMenu()

Supported OS

All
