FinishFTPDirectory()

Syntax

FinishFTPDirectory(#Ftp)
Description
Finishes the enumeration started with ExamineFTPDirectory(). This allows to free the resources associated with the FTP listing.

Parameters

#Ftp The connection to use.

Return value

None.

See Also

ExamineFTPDirectory()

Supported OS

All
