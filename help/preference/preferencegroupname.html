PreferenceGroupName()

Syntax

Group$ = PreferenceGroupName()
Description
Returns the name of the current group being enumerated with ExaminePreferenceGroups().

Parameters

None.

Return value

The name of the current group being enumerated with ExaminePreferenceGroups().
