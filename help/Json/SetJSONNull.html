SetJSONNull()

Syntax

SetJSONNull(JSONValue)
Description
Clear the JSON value and set the type to #PB_JSON_Null.

Parameters

JSONValue The JSON value.

Return value

None.

Example

  ParseJSON(0, "[1, 2, 3, 4, 5]") 
  SetJSONNull(GetJSONElement(JSONValue(0), 2))
  SetJSONNull(GetJSONElement(JSONValue(0), 3))
  Debug ComposeJSON(0)

See Also
