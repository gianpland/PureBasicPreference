ScreenID()

Syntax

Result = ScreenID()
Description
Returns the OS ScreenID.

Parameters

None.

Return value

The OS ScreenID. On Windows this is a normal WindowID so any function which needs such an ID can use it, like PlayMovie() for example.

Supported OS

Windows (DirectX), Linux (SDL)
