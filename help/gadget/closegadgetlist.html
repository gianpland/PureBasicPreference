CloseGadgetList()

Syntax

CloseGadgetList()
Description
Terminate the current gadget list creation and go back to the previous GadgetList.

Parameters

None.

Return value

None.

Remarks

This is especially useful for the following gadgets:

- ContainerGadget()
- PanelGadget()
- ScrollAreaGadget()

See Also

OpenGadgetList(), ContainerGadget(), PanelGadget(), ScrollAreaGadget()

Supported OS

All
