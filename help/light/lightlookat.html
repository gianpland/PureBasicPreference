LightLookAt()

Syntax

LightLookAt(#Light, x, y, z)
Description
Changes the #Light orientation in the world, to point towards the specified x,y,z point.

Parameters

#Light The light use.
x, y, z The point to look at.

Return value

None.

Supported OS

All
