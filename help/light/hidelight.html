HideLight()

Syntax

HideLight(#Light, State)
Description
Hides or shows the specified #Light.

Parameters

#Light The light to use.
State It can be one of following values: 
  #True : the #Light is hidden
  #False: the #Light is shown

Return value

None.

Supported OS

All
