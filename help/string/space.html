Space()

Syntax

Result$ = Space(Length)
Description
Creates a string of the given length filled with 'space' characters.

Parameters

Length The length (in characters) of the new string.

Return value

A new string filled with 'space' characters.

Example

  Debug "-" + Space(5) + "-" ; Will display "-     -"

Supported OS

All
