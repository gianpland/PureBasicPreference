SetXMLEncoding()

Syntax

SetXMLEncoding(#XML, Encoding)
Description
Changes the text encoding used for exporting/saving the given XML tree.

Parameters

#XML The XML to use.
Encoding It can be one of the following values: 
   #PB_Ascii
   #PB_Unicode (UTF16)
   #PB_UTF8
This only affects the exporting/saving of the tree. The data in the #XML object is always stored in the PB string format (Ascii or Unicode depending on the compiler option). So a unicode executable can safely change the encoding to #PB_Ascii for saving and then back to something else without loosing any information in the tree in memory.

Return value

None.

See Also

ExportXML(), SaveXML(), GetXMLEncoding()

Supported OS

All
