NextXMLNode()

Syntax

Result = NextXMLNode(Node)
Description
Returns the next XML node after the given one (inside their parent node).

Parameters

Node The XML node to use.

Return value

The node after the specified node or zero if there are no more nodes.

See Also

PreviousXMLNode()

Supported OS

All
