GetXMLEncoding()

Syntax

Result = GetXMLEncoding(#XML)
Description
Returns the text encoding used for exporting/saving the given XML tree.

Parameters

#XML The XML to use.

Return value

It can be one of the following values: 
   #PB_Ascii
   #PB_Unicode (UTF16)
   #PB_UTF8

See Also

ExportXML(), SaveXML(), SetXMLEncoding()

Supported OS

All
