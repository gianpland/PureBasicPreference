SpriteWidth()

Syntax

Result = SpriteWidth(#Sprite)
Description
Returns the width (in pixels) of the specified sprite.

Parameters

#Sprite The sprite to use.

Return value

The width (in pixels) of the specified sprite.

See Also

SpriteHeight(), SpriteDepth()

Supported OS

All
