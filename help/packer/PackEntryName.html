PackEntryName()

Syntax

Result$ = PackEntryName(#Pack)
Description
Returns the name of the current entry, set with NextPackEntry().

Parameters

#Pack The pack to use.

Return value

Returns the name of the current entry.

See Also

OpenPack(), NextPackEntry(), PackEntryType(), PackEntrySize()

Supported OS

All
