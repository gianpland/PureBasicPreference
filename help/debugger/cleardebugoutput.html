ClearDebugOutput()

Syntax

ClearDebugOutput()
Description
Clear the content of the debug output window.

Parameters

None.

Return value

None.

Example

  ; Show 10 debug values only, not a continuous list
  Repeat
    ClearDebugOutput()
    For i = 1 To 10
      Debug x
      x + 1
    Next i
    
    Delay(500)
  ForEver

See Also

Included debugging tools, Debug, ShowDebugOutput(), SaveDebugOutput()

Supported OS

All
