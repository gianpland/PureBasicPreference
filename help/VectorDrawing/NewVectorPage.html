NewVectorPage()

Syntax

NewVectorPage()
Description
Finishes the current page on the vector drawing output and starts a fresh page.

The following outputs support multiple pages:

PrinterVectorOutput()
PdfVectorOutput()
SvgVectorOutput()

Parameters

None.

Return value

None.

See Also

NewPrinterPage()

Supported OS

All
