IsPathEmpty()

Syntax

Result = IsPathEmpty()()
Description
Tests if the current vector drawing path is empty.

Parameters

None.

Return value

Returns non-zero if the path is empty and zero if the path contains any line segments.

See Also

ResetPath(), IsInsidePath(), IsInsideStroke()

Supported OS

All
