CountString()

语法

Result = CountString(String$, StringToCount$)
概要
Returns the number of occurrences of StringToCount$ found in String$.

参数

String$ The input string to use.
StringToCount$ The string to be counted in the input string.

返回值

The number of occurrences of &apos;StringToCount$&apos; found in &apos;String$&apos;.

备注

The counting is not word based, which means that if the &apos;StringToCount$&apos; is a part of a word, it will be counted as well, as shown in the following example.

示例

  Debug CountString("How many &apos;ow&apos; contains Bow ?", "ow") ; will display 3

已支持操作系统

所有

<- Chr() - String Index - EscapeString() ->
