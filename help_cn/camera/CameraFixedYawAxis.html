CameraFixedYawAxis()

语法

CameraFixedYawAxis(#Camera, Enable [, VectorX, VectorY, VectorZ])
概要
Change the fixed yaw axis of the camera. The default behaviour of a camera is to yaw around its own Y axis.

参数

#Camera The camera to use.
Enable Enable or disable the use of a custom yaw axis. If set to #True, a new axis vector has to be specified. If set to #False, the camera will yaw around its own Y axis.
VectorX (optional) &apos;x&apos; vector direction of the new yaw axis (usually a value between -1.0 and 1.0, if not it will be automatically normalized). &apos;Enable&apos; parameter has to be set to have any effect.
VectorY (optional) &apos;y&apos; vector direction of the new yaw axis (usually a value between -1.0 and 1.0, if not it will be automatically normalized). &apos;Enable&apos; parameter has to be set to have any effect.
VectorZ (optional) &apos;z&apos; vector direction of the new yaw axis (usually a value between -1.0 and 1.0, if not it will be automatically normalized). &apos;Enable&apos; parameter has to be set to have any effect.

返回值

无.

已支持操作系统

所有
