CameraProjectionY()

语法

Result = CameraProjectionY(#Camera, x, y, z)
概要
Returns the &apos;y&apos; position, in pixels, of a 3D point on the specified #Camera. If the point is outside of the camera current view, it returns -1. This is very useful to map 3D points to 2D screen.
