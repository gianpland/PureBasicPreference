ExamineKeyboard()

语法

Result = ExamineKeyboard()
概要
Updates the keyboard state. This function has to be called before using KeyboardInkey(), KeyboardPushed() or KeyboardReleased().

参数

无.

返回值

无.

参阅

KeyboardInkey(), KeyboardPushed() KeyboardReleased().

已支持操作系统

所有
