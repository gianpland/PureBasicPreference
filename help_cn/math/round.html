Round()

语法

Result.f = Round(Number.f, Mode)
概要
Round the specified float number according to the given mode.

参数

Number.f The number to round.
模式 The rounding mode. This can be one of the following values: 
  #PB_Round_Down   : rounds down the number
  #PB_Round_Up     : rounds up the number
  #PB_Round_Nearest: rounds to the nearest integer number (0.5 and up will round up)

备注

To turn a floating-point number into an integer without rounding, use Int() or IntQ(). This function can handle float and double values.

示例

  Debug Round(11.6, #PB_Round_Down)    ; Will print &apos;11&apos;
  Debug Round(-3.6, #PB_Round_Down)    ; Will print &apos;-4&apos;

  Debug Round(11.6, #PB_Round_Up)      ; Will print &apos;12&apos;
  Debug Round(-1.6, #PB_Round_Up)      ; Will print &apos;-1&apos;

  Debug Round(11.6, #PB_Round_Nearest) ; Will print &apos;12&apos;
  Debug Round(11.4, #PB_Round_Nearest) ; Will print &apos;11&apos;
  Debug Round(11.5, #PB_Round_Nearest) ; Will print &apos;12&apos;
  Debug Round(-7.5, #PB_Round_Nearest) ; Will print &apos;-8&apos;
