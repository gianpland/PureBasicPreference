Pow()

语法

Result.f = Pow(Number.f, Power.f)
概要
Returns the given number, raised to the given power.

参数

Number.f The mantissa.
Power.f The exponent. If &apos;Number.f&apos; is negative, the exponent must be a whole number.

返回值

Returns the value of &apos;Number&apos; raised to the power &apos;Power&apos;.

备注

This function can handle float and double values.
The symbol &apos;^&apos; itself is not a supported operator. This is what the Pow() function is available for.

示例

  Debug Pow(2.0, 3.0)  ; will display &apos;8.0&apos;
