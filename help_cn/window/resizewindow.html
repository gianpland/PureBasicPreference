ResizeWindow()

语法

ResizeWindow(#Window, x, y, Width, Height)
概要
Move and resize the given window to the given position and size. If any of the parameters should be ignored (not be changed) #PB_Ignore can be passed at this place.

参数

#Window The window to resize.
x, y The new window position, in pixels. If &apos;x&apos; or &apos;y&apos; is set to #PB_Ignore, the current value of &apos;x&apos; or &apos;y&apos; won&apos;t be changed.
Width, Height The new window size, in pixels. If &apos;Width&apos; or &apos;Height&apos; is set to #PB_Ignore, the current value of &apos;Width&apos; or &apos;Height&apos; won&apos;t be changed.

返回值

无.

已支持操作系统

所有
