FreeToolBar()

语法

FreeToolBar(#ToolBar)
概要
Free the specified #Toolbar.

参数

#ToolBar The toolbar to free. If #PB_All is specified, all the remaining toolbar are freed.

返回值

无.

Remarks

All remaining toolbars are automatically freed when the program ends.
