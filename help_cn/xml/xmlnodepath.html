XMLNodePath()

语法

Result$ = XMLNodePath(Node [, ParentNode])
概要
Returns a string representing the relation between Node and ParentNode.

参数

Node The XML node to get the path.
ParentNode (optional) It has to be a parent or grandparent of &apos;Node&apos;. If omitted, the root node of the tree is used.

返回值

A string representing the relation between &apos;Node&apos; and &apos;ParentNode&apos;. See XMLNodeFromPath() for a description of the returned path string.
