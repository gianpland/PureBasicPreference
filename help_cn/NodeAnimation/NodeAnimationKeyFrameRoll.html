NodeAnimationKeyFrameRoll()

语法

Result = NodeAnimationKeyFrameRoll(#NodeAnimation, KeyFrame)
概要
Returns the #NodeAnimation keyframe roll.

参数

#NodeAnimation The node animation to use.
#Keyframe The node animation keyframe index. The first keyframe index is zero.

返回值

Returns the current node animation keyframe roll, or zero if the keyframe doesn't exists.

参阅

CreateNodeAnimationKeyFrame(), NodeAnimationKeyFramePitch(), NodeAnimationKeyFrameYaw()

已支持操作系统

所有

<- NodeAnimationKeyFramePitch() - NodeAnimation Index - NodeAnimationKeyFrameX() ->
