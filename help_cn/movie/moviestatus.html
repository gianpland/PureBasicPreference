MovieStatus()

语法

Result.q = MovieStatus(#Movie)
概要
Get the playback status of the movie.

参数

#Movie The movie to use.

返回值

Returns one of the following values:

-1: Movie is paused.
0: Movie is stopped
> 0: Movie is playing. The returned value is the current frame number displayed.
