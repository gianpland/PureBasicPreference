MovieWidth()

语法

Width = MovieWidth(#Movie)
概要
Returns the width of the movie.

参数

#Movie The movie to use.

返回值

Returns the width of the movie in pixels. If the result equals -1, then no video stream is found (or isn&apos;t compatible), however the audio stream can still be played.

参阅

MovieWidth(), MovieLength(), MovieInfo()

已支持操作系统

所有
