CloseConsole()

语法

CloseConsole()
概要
使用OpenConsole()打开控制台窗口前请先关闭先前的控制台窗口。Once the console has been closed, it&apos;s not possible to use any console-related functions unless you open the console again. 您的程序结束后控制台窗口会自动关闭。

参数

无.

返回值

无.

示例

  For i = 0 To 4
    If OpenConsole()
      PrintN("这是一个控制台 #"+Str(i))
      PrintN("按回车键关闭控制台")
      Input()
      CloseConsole()
    EndIf
  Next
