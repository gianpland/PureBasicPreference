ConsoleCursor()

语法

ConsoleCursor(Height)
概要
更改光标的显示，该指示器将在设置之后显示字符时开始使用。 . 此函数允许您更改光标的高度。

参数

Height 光标的新高度。光标高度值可以为0(针对隐藏鼠标)或者1至10之间。. 目前有三个值有效： 
  1 : Underline 下划线光标 (默认)
  5 : Mid-height 中高光标
  10: Full-height 完整高度光标

返回值

无.

备注

By default the cursor is a flashing underline in consoles under the Windows OS. Note that you might need to make the console window full-screen to see the true effect of this function.

控制台必须是图形模式, 请参阅 EnableGraphicalConsole().

示例

  If OpenConsole()
    EnableGraphicalConsole(1)

    For CursorHeight = 0 To 10
      ConsoleCursor(CursorHeight)
      PrintN("Press return to increase cursor size")
      Input()
    Next
    
    PrintN("Press return to exit")
    Input()
  EndIf
