ConsoleLocate()

语法

ConsoleLocate(x, y)
概要
使用字符坐标移动光标到指定的位置。 调用这个函数后，任何你要打印的字符都会从这个坐标开始。

参数

x 控制台中的水平位置(从0开始)。
y 控制太重的垂直位置（从0开始）。

返回值

无.

备注

The console has to be in graphical mode, see EnableGraphicalConsole().

示例

  If OpenConsole()
    EnableGraphicalConsole(1)
    
    For i = 0 To 200
      ConsoleLocate(Random(79), Random(24))
      Print("*")
    Next
    
    ConsoleLocate(30, 10)
    PrintN("Press return to exit")
    Input()
  EndIf
