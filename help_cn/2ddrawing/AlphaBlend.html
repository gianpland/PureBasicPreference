AlphaBlend()

语法

Color = AlphaBlend(Color1, Color2)
概要
返回融合了两个32位的色彩所产生的32位色。

参数

Color1 此颜色用于前景色. It is blended on to &apos;Color2&apos;.
Color2 此颜色用于背景色.

返回值

返回组合后的颜色.

备注

The RGBA() function can be used to create 32-bit colors with alpha transparency. 这些函数对于执行绘图操作非常有用。
