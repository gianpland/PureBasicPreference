DatabaseID()

语法

DatabaseID = DatabaseID(#Database)
概要
Returns the unique ID which identifies the given &apos;#Database&apos; in the operating system. This function is useful when another library needs a database reference.

参数

#Database The database to use.

返回值

Returns the ID for this database connection.

已支持操作系统

所有

<- DatabaseError() - Database Index - DatabaseQuery() ->
