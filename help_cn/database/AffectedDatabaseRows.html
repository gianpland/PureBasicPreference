AffectedDatabaseRows()

语法

Result = AffectedDatabaseRows(#Database)
概要
Returns the number of rows affected by the last DatabaseUpdate() operation.

参数

#Database The database to use.

返回值

Returns the number of rows affected by the last DatabaseUpdate() operation.

参阅

DatabaseUpdate()

已支持操作系统

所有
