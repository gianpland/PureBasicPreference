LoadSound3D()

语法

Result = LoadSound3D(#Sound3D, Filename$ [, Flags])
概要
Loads a WAV or an OGG sound file. The sound has to be mono, as stereo sounds doesn&apos;t allow space positioning.

参数

#Sound3D A number to identify the new sound. #PB_Any can be used to auto-generate this number.
Filename$ The sound filename to load.
Flags (optional) It can be the following value: 
  #PB_Sound3D_Streaming: Enable sound streaming playback

返回值

Nonzero if the sound has been successfully loaded, zero otherwise.

备注

A sound doesn&apos;t hold its position. It can be attached to a node object to have its own position. A sound is played relative to the listener location. To change the listener location, use SoundListenerLocate().
