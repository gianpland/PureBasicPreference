TerrainTilePointX()

语法

Result = TerrainTilePointX(#Terrain, TileX, TileY, x, y, z)
概要
Returns the &apos;x&apos; position of the point in the terrain tile.

参数

#Terrain The terrain to use.
TileX The tile &apos;x&apos; index.
TileY The tile &apos;y&apos; index.
x, y, z The position of the point in the world (in world unit).

返回值

Returns &apos;x&apos; position (in pixels) of the point in the terrain tile.

参阅

CreateTerrain(), DefineTerrainTile()

已支持操作系统

所有
