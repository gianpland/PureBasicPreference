EventDropPrivate()

语法

Result = EventDropPrivate()
概要
After a #PB_Event_GadgetDrop or #PB_Event_WindowDrop is received by WaitWindowEvent() or WindowEvent() with a format (can be get with EventDropType()) of #PB_Drop_Private, this function returns the &apos;PrivateType&apos; that was dropped.

参数

无.

返回值

Returns the private type value used when starting the Drag & Drop operation. (See DragPrivate() for more information.)

参阅

EnableGadgetDrop(), EnableWindowDrop(), EventDropType(), EventDropAction(), EventDropX(), EventDropY()

已支持操作系统

所有
