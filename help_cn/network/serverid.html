ServerID()

语法

Result = ServerID(#Server)
概要
Returns the unique system identifier of the server.

参数

#Server The server to use.

返回值

Returns the system identifier. This result is sometimes also known as &apos;Handle&apos;. Look at the extra chapter Handles and Numbers for more information.
