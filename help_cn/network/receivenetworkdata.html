ReceiveNetworkData()

语法

Result = ReceiveNetworkData(Connection, *DataBuffer, DataBufferLength)
概要
Receives raw data from the specified client. This function can be used by both client and server applications and should be called only after having received a #PB_NetworkEvent_Data event.

参数

Connection The connection to receive data from. On the server side &apos;Connection&apos; is the client which has send the data (ClientID can be easily obtained with EventClient()). On client side, &apos;Connection&apos; is returned by OpenNetworkConnection().
*DataBuffer The memory address to receive the data to.
DataBufferLength The length of the buffer to receive data to.

返回值

Returns the number of bytes received. If &apos;Result&apos; is equal to DataBufferLength then more data is available to be read. If an error occurred on the connection (link broken, connection close by the server etc...) &apos;Result&apos; will be -1.

备注

On UDP connections, the maximum &apos;DataBufferLength&apos; is 2048. On TCP connections, the maximum &apos;DataBufferLength&apos; is 65536.
