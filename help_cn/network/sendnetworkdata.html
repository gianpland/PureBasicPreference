SendNetworkData()

语法

Result = SendNetworkData(Connection, *MemoryBuffer, Length)
概要
Sends raw data to the specified client. This function can be used by both client and server applications.

参数

Connection The connection to send data to. On the server side, &apos;Connection&apos; is the client which should receive this data. On the client side, &apos;Connection&apos; is returned by OpenNetworkConnection().
*MemoryBuffer The address of the data to send.
Length The length of the data to send.

返回值

Returns the number of bytes actually sent. If the number of bytes returned is not equal to the &apos;Length&apos; parameter the receiving buffer of the user is probably full. If nothing could be sent then &apos;Result&apos; will equal -1.

备注

On UDP connections, the maximum &apos;Length&apos; is 2048. On TCP connections, the maximum &apos;Length&apos; is 65536.
