CheckFilename()

语法

Result = CheckFilename(Filename$)
概要
Checks if the specified Filename$ doesn&apos;t contain invalid characters for the file-system. For example, on Windows &apos;/&apos; and &apos;\&apos; characters are not allowed in the filename.

参数

Filename$ The filename to check without a path.

返回值

Returns nonzero if the filename does not contain invalid characters and zero if it does.
