UseCRC32Fingerprint()

语法

UseCRC32Fingerprint()
概要
Register the CRC32 fingerprint plugin for future use.

参数

无.

返回值

无.

备注

CRC32 is a 32-bit fingerprint not intended for password storage as it&apos;s easily crackable, but for quick data integrity check. For example, zip files have a CRC32 checksum at the end of each file to be sure that the zip is not corrupted. The main advantage of CRC32 over MD5 or other fingerprint algorithm is its very high speed.

示例

  UseCRC32Fingerprint()

  Debug StringFingerprint("any text", #PB_Cipher_CRC32)
