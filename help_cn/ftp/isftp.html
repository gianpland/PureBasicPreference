IsFtp()

语法

Result = IsFtp(#Ftp)
概要
Tests if the given #Ftp number is a valid and correctly initialized ftp client.

参数

#Ftp The connection to use.

返回值

Returns nonzero if the given number is a valid and correctly initialized client connection.

备注

This function is bulletproof and can be used with any value. If the &apos;Result&apos; is not zero then the object is valid and initialized, else it returns zero. This is the correct way to ensure an ftp handle is ready to use.

参阅

OpenFTP(), CloseFTP(), CheckFTPConnection()
