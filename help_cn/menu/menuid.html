MenuID()

语法

MenuID = MenuID(#Menu)
概要
Returns the unique system identifier of the given menu.

参数

#Menu The menu to use.

返回值

Returns the ID of the menu. This sometimes also known as &apos;Handle&apos;. Look at the extra chapter "Handles and Numbers" for more information.

参阅

CreateMenu(), CreatePopupMenu(), CreateImageMenu(), CreatePopupImageMenu()

已支持操作系统

所有

<- MenuHeight() - Menu Index - MenuItem() ->
