SetMenuItemText()

语法

SetMenuItemText(#Menu, Item, Text$)
概要
Changes the text of the specified menu item.

参数

#Menu The menu to use.
Item The item number of the item to change.
Text$ The new text for the item.

返回值

无.

参阅

GetMenuItemText(), MenuItem()
