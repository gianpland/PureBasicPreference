IsJSON()

语法

Result = IsJSON(#JSON)
概要
Tests if the given #JSON number represents valid and correctly initialized JSON data.

参数

#JSON The JSON to use.

返回值

Nonzero if #JSON is valid JSON data, zero otherwise.

备注

This function is bulletproof and can be used with any value. If the &apos;Result&apos; is not zero then the object is valid and initialized, otherwise it will equal zero.

参阅

CreateJSON(), CatchJSON(), LoadJSON(), ParseJSON(), FreeJSON()
