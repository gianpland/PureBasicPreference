VectorOutputWidth()

语法

Result.d = VectorOutputWidth()
概要
Returns the width of the vector drawing output area.

参数

无.

返回值

Returns the output width.

示例

参阅

VectorOutputHeight(), VectorUnit(), VectorResolutionX(), VectorResolutionY()

已支持操作系统

所有
