VectorOutputHeight()

语法

Result.d = VectorOutputHeight()
概要
Returns the height of the vector drawing output area.

参数

无.

返回值

Returns the output height.

示例

参阅

VectorOutputWidth(), VectorUnit(), VectorResolutionX(), VectorResolutionY()

已支持操作系统

所有
