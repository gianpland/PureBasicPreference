LightDirectionX()

语法

Result = LightDirectionX(#Light [, Mode])
概要
Returns the &apos;x&apos; direction vector of the light.

参数

#Light The light to use.
Mode (optional) The mode to get the &apos;x&apos; direction vector. It can be one of the following value: 
  #PB_Absolute: get the absolute &apos;x&apos; direction vector of the light in the world (default).
  #PB_Relative: get the &apos;x&apos; direction vector of the light relative to its parent.

返回值

The &apos;x&apos; direction vector of the light. This value is always between -1.0 and 1.0.

参阅

LightDirectionY(), LightDirectionZ(), LightDirection()
