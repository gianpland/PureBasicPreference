Break : Continue


语法
Break [Level]
概要
Break provides the ability to exit during any iteration, for the following loops: Repeat, For, ForEach and While. The optional &apos;level&apos; parameter specifies the number of loops to exit from, given several nested loops. If no parameter is given, Break exits from the current loop.

Example: Simple loop

  For k=0 To 10
    If k=5
      Break  ; Will exit directly from the For/Next loop
    EndIf
    Debug k
  Next

Example: Nested loops

  For k=0 To 10
    Counter = 0
    Repeat
      If k=5
        Break 2 ; Will exit directly from the Repeat/Until and For/Next loops
      EndIf
      Counter+1
    Until Counter > 1
    Debug k
  Next

语法
Continue
概要
Continue provides the ability to skip straight to the end of the current iteration (bypassing any code in between) in the following loops: Repeat, For, ForEach, and While.

示例

  For k=0 To 10
    If k=5
      Continue  ; Will skip the &apos;Debug 5&apos; and go directly to the next iteration
    EndIf
    Debug k
  Next
