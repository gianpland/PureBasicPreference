SoundVolume()

语法

SoundVolume(#Sound, Volume [, Channel])
概要
Change the sound volume, in real-time.

参数

#Sound The sound to use. If #PB_All is specified, all the sounds (and all channels) are affected.
Volume The new volume for the sound. Valid values are from 0 (no volume) to 100 (full volume).
Channel (optional) The channel to use. It&apos;s the value returned by PlaySound() when using the #PB_Sound_MultiChannel flag. If #PB_All is specified, all the channels of the sound are affected.

返回值

无.

示例

 InitSound()           ; Initialize Sound system
 UseOGGSoundDecoder()  ; Use ogg files

 ; Loads a sound from a file
 LoadSound(0, #PB_Compiler_Home +"Examples\3D\Data\Siren.ogg")
 ; The sound is playing
 PlaySound(0, #PB_Sound_Loop, 20)
  
 MessageRequester("Info", "The sound volume is at 20%")
 
 SoundVolume(0, 80)
 MessageRequester("Info", "The sound volume is at 80%")
 
 FreeSound(0) ; The sound is freed
 End
