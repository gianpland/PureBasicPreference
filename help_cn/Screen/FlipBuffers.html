FlipBuffers()

语法

FlipBuffers()
概要
Flip the back and front buffers of the current screen. The invisible area is now visible and vice versa, which allowss to do a &apos;double-buffering&apos; effect (flicker free graphical displays). A screen must have been opened with OpenScreen() or OpenWindowedScreen(). The way the buffer are flipped (with or without synchronization) is set by OpenScreen() or OpenWindowedScreen().

参数

无.

返回值

无.

备注

FlipBuffers() has to be called outside a StartDrawing() : ... : StopDrawing() program block.

参阅

OpenScreen(), OpenWindowedScreen()

已支持操作系统

所有
