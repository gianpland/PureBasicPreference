AvailableScreenMemory()

语法

Result = AvailableScreenMemory()
概要
Returns the available video memory for the sprites and other video data (in bytes). This function must be called after OpenScreen() or OpenWindowedScreen().

参数

无.

返回值

The available video memory for the sprites and other video data (in bytes).

参阅

OpenScreen(), OpenWindowedScreen()

已支持操作系统

Windows (DirectX), Linux (SDL)
