FinishMesh()

语法

FinishMesh(Mode)
概要
Finish the creation of the current mesh started with CreateMesh().

参数

模式 If set to #True, the mesh will be converted to a static mesh and will be not modifiable anymore. If set to #False, the mesh will still be modifiable with UpdateMesh(). Static meshes are faster to render than dynamic meshes.

返回值

无.
