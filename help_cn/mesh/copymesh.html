CopyMesh()

语法

Result = CopyMesh(#Mesh, #NewMesh)
概要
Creates a #NewMesh which is the exact copy of the specified #Mesh. If #PB_Any is used as the &apos;#NewMesh&apos; parameter, then the new mesh number will be returned as &apos;Result&apos;. Dynamic meshes are not supported for copy (meshes created with the #PB_Mesh_Dynamic flag).

If the Result equals 0, then the mesh copy has failed. If #NewMesh already exists, then it is freed and replaced by a new one.
