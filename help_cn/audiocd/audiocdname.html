AudioCDName()

语法

Result$ = AudioCDName()
概要
Returns the system-dependent name associated with the current audio CD drive.

参数

无.

返回值

Returns a string containing the name. For example, it would return &apos;D:\&apos; on a Windows-based system, if the audio CD is associated as drive letter &apos;D&apos;.

参阅

UseAudioCD()

已支持操作系统

所有
