EntityLinearFactor()

语法

EntityLinearFactor(#Entity, x, y, z)
概要
Changes the linear factor for the #Entity. When moved, the entity linear velocity is multiplied by the linear factor to get the final velocity. This is very useful to constraint an entity move on one or several axis. By default, the linear factor is 1 for all axis meaning no impact on the velocity. The entity needs a physic body to support linear velocity constraint.

参数

#Entity The entity to use.
x The &apos;x&apos; linear factor value. If set to the 0, the entity won&apos;t be able to move on the &apos;x&apos; axis anymore.
y The &apos;y&apos; linear factor vector. If set to the 0, the entity won&apos;t be able to move on the &apos;y&apos; axis anymore.
z The &apos;z&apos; linear factor vector. If set to the 0, the entity won&apos;t be able to move on the &apos;z&apos; axis anymore.

返回值

无.

已支持操作系统

所有
