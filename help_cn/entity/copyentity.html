CopyEntity()

语法

Result = CopyEntity(#Entity, #NewEntity)
概要
Creates a #NewEntity which is the exact copy of the specified #Entity. If #PB_Any is used as &apos;#NewEntity&apos; parameter, the new entity number will be returned as &apos;Result&apos;.

If the &apos;Result&apos; is 0, the entity copy has failed. If #NewEntity was already created, it will be automatically freed and replaced by the new one.
