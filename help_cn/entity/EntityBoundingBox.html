EntityBoundingBox()

语法

Result = EntityBoundingBox(#Entity, Flags)
概要
Returns the position of the bounding box, either in local or world coordinate.

参数

#Entity The entity to use.
Flags Flags can be one of the following values: 
  #PB_Entity_MinBoundingBoxX: Min &apos;x&apos; position of the bounding box
  #PB_Entity_MaxBoundingBoxX: Max &apos;x&apos; position of the bounding box
  #PB_Entity_MinBoundingBoxY: Min &apos;y&apos; position of the bounding box
  #PB_Entity_MaxBoundingBoxY: Max &apos;y&apos; position of the bounding box
  #PB_Entity_MinBoundingBoxZ: Min &apos;z&apos; position of the bounding box
  #PB_Entity_MaxBoundingBoxZ: Max &apos;z&apos; position of the bounding box
combined with one of the following values: 
  #PB_Entity_WorldBoundingBox: Positions are returned in world coordinates (default)
  #PB_Entity_LocalBoundingBox: Positions are returned in local coordinates

返回值

Returns the position of the bounding box, either in local or world coordinate.
