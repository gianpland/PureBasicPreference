EntityFixedYawAxis()

语法

EntityFixedYawAxis(#Entity, Enable [, VectorX, VectorY, VectorZ])
概要
Change the fixed yaw axis of the entity. The default behaviour of a entity is to yaw around its own Y axis.

参数

#Entity The entity to use.
Enable Enable or disable the use of a custom yaw axis. If set to #True, a new axis vector has to be specified. If set to #False, the entity will yaw around its own Y axis.
VectorX (optional) X vector direction of the new yaw axis (value between -1.0 and 1.0). &apos;Enable&apos; parameter has to be set to have any effect.
VectorY (optional) Y vector direction of the new yaw axis (value between -1.0 and 1.0). &apos;Enable&apos; parameter has to be set to have any effect.
VectorZ (optional) Z vector direction of the new yaw axis (value between -1.0 and 1.0). &apos;Enable&apos; parameter has to be set to have any effect.

返回值

无.

已支持操作系统

所有
