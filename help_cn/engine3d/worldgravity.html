WorldGravity()

语法

WorldGravity(Gravity.f)
概要
Changes the gravity of the world when the physics engine is enabled with EnableWorldPhysics().

参数

Gravity The new gravity to set. The default gravity value is set to -9.806 (which is Earth&apos;s gravitational value).

返回值

无.

参阅

EnableWorldPhysics()
