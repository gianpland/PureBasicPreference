NodeY()

语法

Result = NodeY(#Node [, Mode])
概要
Returns the &apos;y&apos; position of the node in the world.

参数

#Node The node to use.
Mode (optional) The mode to get the &apos;y&apos; position. It can be one of the following value: 
  #PB_Absolute: get the absolute &apos;y&apos; position of the node in the world (default).
  #PB_Relative: get the &apos;y&apos; position of the node relative to its parent.

返回值

The &apos;y&apos; position of the node in the world.

参阅

NodeX(), NodeZ()

已支持操作系统

所有
