BillboardGroupZ()

语法

Result = BillboardGroupZ(#BillboardGroup [, Mode])
概要
Determines the absolute position of the billboard group in the world.

参数

#BillboardGroup The number of the billboard group to get the position of.
Mode (optional) The mode to get the &apos;z&apos; position. It can be one of the following value: 
  #PB_Absolute: get the absolute &apos;z&apos; position of the billboard group in the world (default).
  #PB_Relative: get the &apos;z&apos; position of the billboard group relative to its parent.

返回值

The absolute &apos;z&apos; position of the billboard group.

已支持操作系统

所有

<- BillboardGroupY() - Billboard Index - BillboardHeight() ->
