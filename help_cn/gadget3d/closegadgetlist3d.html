CloseGadgetList3D()

语法

CloseGadgetList3D()
概要
Closes the current gadget list creation and go back to the previous GadgetList.

参数

无.

返回值

无.

备注

This function is supported by the following 3D gadgets:

- ContainerGadget3D()
- PanelGadget3D()
- ScrollAreaGadget3D()

已支持操作系统

所有

<- ClearGadgetItems3D() - Gadget3D Index - ComboBoxGadget3D() ->
