HideGadget3D()

语法

HideGadget3D(#Gadget3D, State)
概要
Hide or show a 3D gadget.

参数

#Gadget3D The 3D gadget to use.
State The new state of the 3D gadget. If State = 1, the gadget will be hidden, if State = 0 it will be shown.

返回值

无.

参阅

DisableGadget3D()

已支持操作系统

所有
