ReadPreferenceLong()

语法

Result = ReadPreferenceLong(Key$, DefaultValue)
概要
Try to read the specified associated &apos;Key$&apos; value.

参数

Key$ The key to read the value from. If PreferenceGroup() has been used, then the search is restricted to the current group.
DefaultValue The default value to return if the key isn&apos;t found or the preference file haven&apos;t been opened correctly (file missing for example).

返回值

The value associated to the specified key, as a long number. if the key isn&apos;t found the default value is returned.

示例

See the example at ReadPreferenceDouble()

参阅

ReadPreferenceDouble(), ReadPreferenceFloat(), ReadPreferenceInteger(), ReadPreferenceQuad(), ReadPreferenceString()
